# impact_of_ai_on_human_mental_growth
Excited to share one of my recent Power BI dashboard projects, where I analyzed the relationship between Artificial Intelligence usage and human mental well-being using an interactive dataset of 52,000 records.
This project focuses on uncovering patterns between AI adoption, creativity, education, profession, and overall well-being through data visualization and interactive analytics.

🔍 Key Dashboard Metrics
📌 Total Records Analyzed: 52,000
📌 Average Well-being Score: 62.87
📌 Average AI Usage Score: 24.57
📌 Average Well-being Risk Flag: 0.13
